# eonet-visualizer

# [Deployed snapshot of master](https://vladkoisnych.github.io/eonet-visualizer/)
---
A small react-three-fiber project to visualize real-time and historical NASA EONET API data on a 3D globe. 
It uses a decoupled time loop to smoothly scrub through 180 days of global events like wildfires, severe storms, and volcanoes without dropping frames. 
An adaptive tracking camera allows you to automatically lock onto specific disaster markers for close-up inspection from orbit.
